## 🚀 Quick Start

To get a local copy up and running, follow these simple steps.

### Prerequisites
* A modern web browser
* Git installed on your local machine

### Installation

1. **Clone the repo**
   ```sh
   git clone [https://github.com/imranbru99/vibe-coding-portfolio-website.git](https://github.com/imranbru99/vibe-coding-portfolio-website.git)

```

2. **Navigate into the directory**
```sh
cd vibe-coding-portfolio-website

```


3. **Open the project**
Open `index.html` (or your entry file) in your preferred browser, or run a local server:
```sh
npx serve .

```
